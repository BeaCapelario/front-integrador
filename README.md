Este repositório contém o desenvolvimento do front-end para um projeto integrador. O objetivo principal foi a criação de uma interface de usuário completa que se conecta a serviços de back-end, consolidando o aprendizado em consumo de APIs, gerenciamento de rotas e estados globais.

---

### O que foi explorado:
* Integração com API: Consumo de endpoints REST utilizando bibliotecas como Axios ou Fetch.
* Gerenciamento de Rotas: Navegação entre diferentes páginas da aplicação com React Router.
* Hooks e Context API: Gerenciamento de estados globais para compartilhamento de informações entre componentes.
* UX/UI Design: Implementação de layouts focados na experiência e facilidade de uso do usuário final.
* Sincronização de Dados: Tratamento de requisições assíncronas e exibição de estados de carregamento e erro.

---

### Tecnologias Utilizadas
| Tecnologia | Descrição |
| :--- | :--- |
| React.js | Biblioteca para construção da interface de usuário. |
| JavaScript | Lógica de programação e integração com serviços. |
| Axios | Cliente HTTP para consumo de APIs de back-end. |
| CSS Modules / Styled Components | Metodologias para estilização de componentes. |
